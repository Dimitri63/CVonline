On considère un système publicitaire réparti sur tous les arrêts d’un réseau de transport en commun.
Ce système sera nommé « PiCom » pour Pi Communication car il repose sur l’utilisation de Raspberry Pi.
Sur chaque arrêt du réseau de transport sont installés :

  • un écran 32 pouces
  • un Raspberry Pi 4 modèle B sur lequel Raspberry Pi OS est installé
  • une clé 4G qui se connecte à un réseau 4G. On supposera que cette clé donne une adresse IP statique à chaque Raspberry Pi
  • un coffret sécurisé et hermétique dans lequel les équipements ci-dessus sont placés
Entre 6h et 20h, le Raspberry Pi de chaque arrêt du réseau de transport affiche des annonces publicitaires.

Picom Client

Pour les clients, il est nécessaire de pouvoir :

  • s’inscrire en précisant son nom, son prénom, un email, un mot de passe contenant au minimum 8 caractères et son numéro de téléphone (l’inscription est gratuite)
  • s’authentifier
  • voir toutes ses annonces publicitaires
  • ajouter une annonce publicitaire. Pour ce faire, l’application propose dans cet ordre
    • de téléverser un fichier PNG ou de rédiger son annonce dans un éditeur HTML intégré à l’application
    • de choisir sur une carte du réseau de transport la ou les zones sur lesquelles il/elle souhaite que son annonce soit diffusée
    • de choisir la ou les tranches horaires de diffusion
    • de choisir la date à laquelle la diffusion doit commencer et le nombre de jours de diffusion
    • de payer en ligne l’annonce en précisant son numéro de carte de crédit, la date d’expiration de la carte ainsi que son CVV
  • prolonger la diffusion d’une annonce (cette prolongation fera l’objet d’un autre paiement)
  • voir les diffusions de chacune de ses annonces publicitaires
  • se déconnecter

Picom Admin

Pour les administrateurs, il est nécessaire de pouvoir :

  • s’authentifier
  • gérer les zones (ajout, modification, suppression)
  • gérer les arrêts (ajout, modification, suppression)
  • gérer les tranches horaires (ajout, modification, suppression)
  • gérer les tarifs de diffusion (ajout, modification, suppression)
  • voir les clients
  • voir les annonces
  • se déconnecter

Développement de l'API

Technologie

Pour le développement de l'API, j'ai choisi d'utiliser JAVA

Pour le développement de mon API de l’application Picom, j’ai choisi d’utiliser le langage JAVA.
JAVA est pour moi un excellent langage pour le développement d’une API. Certes, c’est un langage lourd à mettre en place, mais présente des avantages comme le multithreading, l’extensibilité, la gestion de la mémoire et une haute sécurité.
API est également un langage populaire, utilisé à grande échelle dans le monde entier.
Pour réaliser l’API, j’ai combiné JAVA avec le framework SPRING.
SPRING permet d’ajouter de la sécurité avec SpringSecurity, mais également beaucoup d'avantages comme la gestion des dépendances (SpringWeb, Spring Data JPA, SpringSecurity), l’utilisation d’une architecture MVC et facilite le développement d’API avec spring Boot.
Grâce à Spring Data JPA, l'interaction avec la base de données se fait très simplement à l’aide de drivers importés.
La sécurité est accru avec SpringSecurity
. Et enfin, la gestion des requêtes est très simple grâce à l'excellent ORM Hibernate qui permet de créer des requêtes HQL, des requêtes par dérivation etc…

Développement de Picom Client

Technologie

Pour l’application Client, j’ai choisi utiliser Angular.
Grâce à TypeScript, Angular offre un très bon support robuste et fiable, de plus, Angular respecte parfaitement le modèle MVC (Model-View-Controller) avec ses modules et composants.
Angular prend en charge le développement d'applications à page unique, ce qui rend la navigation plus fluide et donc plus agréable.

Développement de Picom Admin

Technologie

Pour le développement de l’application d’administration Picom, j’ai choisi d’utiliser ElectronJS.
ElectronJS est un framework multiplateforme basé sur JavaScript, HTML et CSS. De ce fait, il ne nécessite pas de connaissance supplémentaire pour pouvoir créer une application de bureau. Depuis la sortie du moteur JavaScript V8, ElectronJS est de plus en plus utilisée par les entreprises.
