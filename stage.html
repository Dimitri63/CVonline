Information relative au stage

Date:

Du 08 octobre 2021 au 07 février 2022

Durée du stage : 4 mois

Travail demandé

Pour mener à bien ce projet, vous devez utiliser un outil de gestion de projet et un outil de versioning.

    Le projet doit inclure les phases suivantes :
  • Maquettage
  • Modélisation UML et/ou Merise
  • Conception d’une base de données
  • Conception et développement des applications Web sans l’utilisation de framework
  • Conception et développement des applications Web avec l’utilisation de framework(s)
  • Écriture des tests (unitaires et d’intégration) pour les deux applications
  • Conception et développement d’une application Android et/ou Electron
  • Conception et développement d’une API REST
